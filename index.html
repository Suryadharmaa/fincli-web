// capabilities

A provider fallback chain with granular reliability labels, persistent metrics in SQLite, and an AI Grounding Guard — built for serious workflows.

// install

Up and running in seconds

Requires Python 3.11+. Choose the global npm wrapper, a local dev install, or set API keys from inside FinCLI.

npm install -g @drico2008/fincli
fincli

Creates an isolated .npm-python environment inside the installed package.

python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
fincli

Editable install with dev extras for contributing.

/ai_model key groq <api_key>
/news_model key finnhub <api_key>
/news_model key twelvedata <api_key>

Keys are stored locally in ~/.fincli/secrets.env and masked in terminal output.

// research engine v3

One command, a complete picture

/research returns a compact, source-aware output with Snapshot, Signal, Risk, Context Blend, Trust Gate, Source Quality, Decision Points, and cited Sources. Modes: --snapshot, --deep, --report.

// trading safety layer

Trade with guardrails

Risk guard with max position size, daily loss limit, kill switch, and leverage warning. Paper trading engine with stop-limit orders, cancel, positions aggregation, and daily PnL. Immutable order audit log — never UPDATE, never DELETE.

// backtesting & portfolio analytics

Test before you commit

Professional backtesting with fees, slippage, spread modeling, walk-forward split, position sizing, 5 strategies, Sharpe/Sortino/Calmar ratios, and Monte Carlo robustness. Portfolio time-series snapshots, rebalancing suggestions, and benchmark comparison.
